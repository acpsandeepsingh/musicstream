# **App Name**: HarmonyStream

## Core Features:

- YouTube Video/Audio Player: Play videos and audio directly. Show all songs by default, not from a URL.
- Background Playback: Allow audio to continue playing even when the browser is minimized or another tab is active.
- YouTube Search Integration: Enable users to search for songs and videos directly from YouTube.
- Top 50 Default Page: Display a curated list of the top 50 trending songs or videos on the home page.
- Playlist Creation: Allow users to create and manage custom playlists of songs and videos.
- User Accounts: Allow users to create accounts to save playlists and preferences.
- Recommendations: Suggest songs based on listening history and current trends using AI.
- Genre Filtering: Allow users to filter songs by genre.
- Dark/Light Mode: Allow users to switch between a dark and light theme.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a modern and musical feel.
- Background color: Light gray (#F5F5F5) to ensure good contrast and readability.
- Accent color: Teal (#009688) to highlight interactive elements.
- Headline font: 'Poppins' (sans-serif) for a modern and clean aesthetic.
- Body font: 'PT Sans' (sans-serif) for readability.
- Use minimalist and consistent icons for navigation and controls.
- Clean and intuitive layout with a focus on easy navigation and content discovery.
- Subtle transitions and animations to enhance the user experience.
